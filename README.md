## 2023-11-06 20:03
### <b>Level_Design.xlsx</b>
수정 완료 되지 않아 버전 1.0 현행 유지
```
- 유닛마다 1단계, 3단계, 5단계에 기본 능력치 [공격력, 공격범위, 공격속도] 현재 데이터로 입력 (Unit_System.pptx)
1단계은 능력치가 최초로 생성되는 단계이며 프레스티지 효과만 적용 가능
3단계는 3단계 해금까지 적용할 수 있는 단계
5단계는 모든 효과가 적용될 수 있는 최종 단계
밸런싱을 위해 1, 3, 5단계를 기준으로 잡고 레벨디자인 후 2단계와 4단계를 마지막에 밸런싱 예정

- Unit 시트 전체 '조건부 서식' 지정.
텍스트 '-' 사용 시 <굵음, 어두운빨간색>

- 라운드보상 3레벨, 5레벨, 프레스티지 효과 셀 한정 '조건부 서식' 지정.
텍스트 영문 'O' 사용 시 <굵음, 연두색>

- 라운드보상 3레벨 해금, 5레벨 해금과 프레스티지 효과 셀들에 '데이터 유효성 검사' 적용 (목록)
드롭다운 목록으로 텍스트 ' - , O ' 중 선택할 수 있음.
'-'는 미적용 상태, 'O'는 적용 상태를 뜻함.
3레벨 해금은 3~5단계만 드롭다운 목록 적용.
5레벨 해금은 5단계만 드롭다운 목록 적용.
프레스티지 효과는 1~5단계만 드롭다운 목록 적용.
선택에 따라서 행의 '최종 능력치' 데이터가 식에 의해 자동으로 계산이 되게끔 제작될 예정.

- Kali 프레스티지 수식 생성
프레스티지 효과 = 공격속도 50% 증가. (1~5레벨 셀 적용)
[ = F5 + IF( I5 = "O", F5 * 0.5, 0 ) ] 식 적용

- Kali 3레벨 해금 수식 생성
3레벨 해금 효과 = 공격 범위 20% 증가 (3~5레벨 셀 적용)
[ = E7 + IF( G7 = "O", E7 * 0.2, 0) ] 식 적용

- Kali 5레벨 해금 수식 생성
5레벨 해금 효과 = 공격 범위 30% 추가 증가 (5레벨 셀 적용)
[ = E9 + IF( G9 = "O", E9 * 0.2, 0) + IF( H9 = "O", E9 * 0.3, 0) ] 식 적용

- '별도 능력치' 열 생성
유키부터 효과에 따라 변경되는 능력치가 생김으로 열 추가.
```
